About

About me

I am a computer science graduate student at NC State working at the intersection of software engineering, machine learning, and AI-assisted development.

I grew up as a software engineer by training, completing my B.S. in Software Engineering at Sharif University of Technology, where I graduated as valedictorian (4.0/4.0 GPA). During my undergraduate years I worked in the Intelligent Systems Lab on code analysis and technical debt, including a transformer-based code smell detector evaluated on a 10K-method Java corpus.

At NC State, I am a graduate research assistant in the Real-World AI / SE Lab advised by Dr. Tim Menzies. My current work focuses on SE-ADAPT, a defect prediction framework that incorporates LLM provenance signals (prompt entropy, generation churn, acceptance rates) to better understand when AI-generated code is likely to fail in production.

I like building tools and datasets that make modern "vibe coding" workflows more honest about complexity, quality, and authorship. Long term, I am excited about research-oriented roles in academia or industry labs that sit between developer tooling, ML, and empirical software engineering.
