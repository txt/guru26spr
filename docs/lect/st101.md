# Object tutorial

### Object and String Utilities

* 
**Stringification:** The `s` method is added to `Object` to return the object's string representation using `printString`.



```smalltalk
! Object methods !
s
  "Return self as a string."
  ^self printString !

```

* 
**Console Output:** The `oo` method prints the object to the `Transcript` (the console) followed by a carriage return.



```smalltalk
! Object methods !
oo
   "Show self on Transcript"
   Transcript nextPutAll: self s;
              cr!!

```

* 
**File Loading Shortcuts:** Both `Symbol` and `String` get a `talk` method, which acts as a shorthand for reading in Smalltalk `.st` files using `FileStream fileIn:`.



```smalltalk
! Symbol methods !
talk
  "Shorthand for reading in code"
  ^FileStream fileIn: self asString,'.st' !

! String methods !
talk
  "Shorthand for reading in code"
  ^FileStream fileIn: self,'.st' !

```

* 
**String Parsing & Padding:** The `asWords` method splits a string into an array based on whitespace. The `right:`, `right:with:`, `left:`, and `left:with:` methods allow you to easily pad strings with spaces or specific characters to a desired length.



```smalltalk
! String methods !
asWords
  "Returns self, divided on whitespace."
^(self tokenize: '\s+') asArray !

right: n
  "Returns self, padded right with spaces."
^self right: n with: $  !

right: n with: char
  "Returns self, padded right with character char."
  |out|
out := WriteStream on:''.
  1 to: n - self size do: [:i| out nextPut: char ].
  out nextPutAll: self.
^out contents !

left: n
  "Returns self, padded left with spaces."
  ^self left: n with: $  !

left: n with: char
  "Returns self, padded left with character char."
  |out s|
  out := WriteStream on: ''.
out nextPutAll: self.
  1 to: n - self size do: [:i| out nextPut: char   ].
  ^out contents !

```

* 
**Unit Tests:** The `String` class is given a `testing` category containing methods like `goodAsWords`, `goodLeft`, and `goodRight` that return boolean values to verify the padding logic works.



```smalltalk
! String class methodsFor: 'testing' !
goodAsWords
   ^'i want candy' asWords = #('i' 'want' 'candy') !
goodLeft
   ^('123' left: 10)  = '123       ' !
goodRight
   ^('123' right: 10) = '       123' !
!

```

### Metaprogramming and Class Generation

* 
**Automatic Getters and Setters:** The `public` class method is a powerful piece of metaprogramming. It takes the instance variables of a class, dynamically writes Smalltalk code for a getter and a setter, and compiles them directly into the class dictionary on the fly.



```smalltalk
! Class methods !
public
  "Auto-create get and set methods for all instance vars."
  |getter setter|
self instanceVariableString asWords do: [:what |
       getter := '%1 [
         "Answer the receiver''s %1."
^%1 ]' % {what} .
       setter := '%1: aValue [
         "Set the receiver''s %1."
%1 := aValue ]' % {what} .
       self compile: getter;
            compile: setter.  ] !

```

* 
**Subclassing Shorthand:** The `sub:kid`, `sub:kid has:vars`, and `sub:kid has:vars1 shares:vars2` methods simplify Smalltalk's normally verbose subclassing syntax. They automatically call standard inheritance methods and immediately chain the `public` method to auto-generate accessors for all new variables.



```smalltalk
! Class methods !
sub: kid  
   "Shorthand for sub-classing."
   ^self sub: kid has: ''  shares: '' !

sub: kid has: vars 
   "Shorthand for sub-classing."
   ^self sub: kid has: vars shares: ''!

sub: kid has: vars1 shares: vars2
   "Shorthand for sub-classing."
^(self
     subclass: kid
     instanceVariableNames: vars1
     classVariableNames: vars2
     poolDictionaries: ''
     category: nil) 
     public !

```

### The "Magic" Base Class

* 
**Auto-initialization:** The `Magic` class overrides the standard `new` method to automatically call an `init` method upon instantiation.



```smalltalk
"All Magic methods always run `init` on new instances
 (so subclasses of Magic that implement `init` get it
 called automatically)."
Object sub: #Magic!

! Magic class methods !
new
  ^super new init !!

! Magic methods !
init !!

```

* 
**The Employee Subclass:** `Employee` inherits from `Magic` and uses the custom `init` method to set default values (like name, age, and shoesize). It also implements a custom `printOn:` method for clean formatting.



```smalltalk
Magic sub: #Employee has: 'name age shoesize'

! Employee methods !
init
  self name: 'freda';
       age:  21;
shoesize:  0 !

printOn: aStream
  aStream 
     nextPutAll: 'Emp(';
     nextPutAll: ':name ',name s;
     nextPutAll: ' :age ',age s;
     nextPutAll: ' :shoesize ',shoesize s;
     nextPutAll: ')' !
!

```

### The "Visit" Polymorphism Pattern

* 
**Generic Visiting:** The `visit:` method is implemented across multiple classes to allow deep traversal of objects. On a standard `Object`, it simply evaluates the block with itself. On a `Collection`, it iterates through its items and calls `visit:` on each one.



```smalltalk
! Object methods !
visit: aBlock
  "Visit one thing"
  ^aBlock value: self !!

! Collection methods !
visit: aBlock
  "Visit a collection of thigns"
  self do: [:x|
    x visit: aBlock]!!

```

* 
**String Override:** `String` overrides `visit:` to evaluate itself as a whole object rather than iterating through its characters.



```smalltalk
! String methods !
visit: aBlock
  aBlock value: self !!

```

* 
**Reflective Visiting:** The `Magic` class uses Smalltalk's reflection (`instSize`, `basicSize`, `instVarAt:`) to iterate over its own internal instance variables and visit them.



```smalltalk
! Magic methods !
visit: aBlock
  "To heck with encapulation. Walk over the instance vars."
  | num |
num := self class instSize + self basicSize.
  1 to: num do: [:i | 
     (self instVarAt: i) visit: aBlock ]
!!

```

### Collection Iterators

* 
**`b4Now:`:** A custom iterator on `Collection` that yields the previous item (`b4`) and the current item (`now`) to a block, which is great for comparing adjacent elements.



```smalltalk
! Collection methods !
b4Now: block
  | b4 |
  self do: [:now|
    b4 notNil ifTrue: [block value: b4 value: now].
b4 := now] !

```

* **`eject:`:** A logical inverse of `select:`. It returns elements that do *not* match the block condition.



```smalltalk
eject: aBlock
  ^self select: [:x| (aBlock value: x) not ] !!

```
